# 软件安装助手 (SoftManager)

一款功能强大的软件批量安装工具，支持静默安装、智能参数检测和方案管理。

## 主要功能

### 🔍 软件扫描与管理
- **智能扫描**：自动扫描指定目录下的所有安装程序
- **README支持**：通过 readme.txt 文件自动加载软件描述信息
- **批量编辑**：双击列表项直接编辑软件信息
- **重复检测**：基于安装路径的唯一索引，防止重复添加

### 🚀 自动安装
- **智能安装**：自动检测和尝试静默安装参数
- **静默参数支持**：为每个软件配置自定义静默安装参数
- **无人值守**：配置静默参数后无需人工干预，自动完成安装
- **批量处理**：一次选择多个软件进行自动安装

### 📋 方案管理
- **方案保存**：将软件组合保存为安装方案
- **方案加载**：快速加载已保存的安装方案
- **方案删除**：管理和清理不需要的安装方案

### ⚙️ 配置管理
- **目录配置**：通过 appsettings.json 配置扫描目录
- **默认配置**：自动生成默认配置文件
- **路径支持**：支持中文和英文目录名称

### 🎨 界面功能
- **直观界面**：清晰的列表视图，显示软件关键信息
- **实时编辑**：双击字段直接编辑，无需打开新窗口
- **状态显示**：实时显示安装进度和状态
- **关于窗口**：软件版本和信息展示

## 安装与配置

### 系统要求
- Windows 7 或更高版本
- .NET 6.0 或更高版本

### 配置文件
软件启动时会在同目录下生成 `appsettings.json` 文件，可根据需要修改：

```json
{
  "ScanDirectory": "..\\常用软件"
}
```

- **ScanDirectory**：指定软件扫描目录，支持相对路径和绝对路径
- 默认值：`..\\常用软件`（相对于可执行文件的父目录）

### README文件格式

在扫描目录中创建 `readme.txt` 文件，采用两行一组的格式：

```
# 软件安装程序文件名
软件描述信息

# 示例
QQPinyin_Setup_6.5.6103.400.exe
拼音输入法，好用的输入法

SogouWB_Setup_3.1.0.1751.exe
搜狗五笔输入法
```

- **支持编码**：UTF-8、GB2312、GBK、UTF-16
- **匹配规则**：支持直接匹配、大小写不敏感、版本号智能匹配

## 使用流程

1. **准备软件**：将软件安装程序放置在配置的扫描目录中
2. **创建README**：在相应目录创建 `readme.txt` 文件，添加软件描述
3. **扫描软件**：运行 SoftManager，点击"扫描新增软件"按钮
4. **审核编辑**：在审核窗口中检查和编辑软件信息
5. **确认添加**：点击"确认添加"按钮，将软件添加到主列表
6. **配置参数**：为需要静默安装的软件配置静默参数
7. **自动安装**：选择软件，点击"自动安装"按钮开始批量安装

## 静默安装参数

### 配置方法
1. 在主界面中双击软件的"静默安装参数"列
2. 输入适用于该软件的静默安装参数
3. 保存后，自动安装时会优先使用配置的参数

### 常见静默参数
- **MSI安装包**：`/qn /norestart`
- **EXE安装包**：`/S` 或 `-s`
- **Inno Setup**：`/verysilent /norestart`
- **NSIS**：`/S`

### 自动检测
当未配置静默参数时，软件会自动尝试常见的静默参数组合，确保最大程度的自动化安装。

## 方案管理

### 保存方案
1. 选择需要包含在方案中的软件
2. 点击"保存方案"按钮
3. 输入方案名称并确认

### 加载方案
1. 从"方案"下拉菜单中选择已保存的方案
2. 软件列表会自动更新为方案中的软件

### 删除方案
1. 从"方案"下拉菜单中选择需要删除的方案
2. 点击"删除方案"按钮
3. 确认删除操作

## 故障排除

### 扫描问题
- **找不到软件**：检查扫描目录配置是否正确
- **README未加载**：确保文件名正确为 `readme.txt`，编码格式支持
- **描述未显示**：检查README文件格式是否为两行一组

### 安装问题
- **安装失败**：检查安装程序是否完整，尝试手动运行测试
- **静默参数无效**：尝试使用不同的静默参数组合
- **权限不足**：以管理员身份运行 SoftManager

### 配置问题
- **配置文件丢失**：软件会自动生成默认配置文件
- **路径错误**：确保路径使用反斜杠 `\\`，支持相对路径
